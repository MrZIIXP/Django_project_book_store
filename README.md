# Это гайд по работе с проэктом
1. Установка зависимостей 
```bash
pip install -r requirements.txt
```
2. Создаём .env файл и добавляем в него:
```bash
EMAIL_USER = 
EMAIL_APP_PASSWORD = 
EMAIL_PORT = 
EMAIL_HOST =

NAME_DB = Lesson_12
```
3. Запуск проекта
```bash	
python manage.py runserver
```
