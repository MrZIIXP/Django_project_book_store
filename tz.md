8. QuerySet, filter и search

На странице списка книг нужно реализовать:

вывод всех книг;

поиск по названию книги;

фильтр по категории;

фильтр по автору;

исключение ненужных данных через exclude();

получение одного объекта через get() или get_object_or_404().

Нужно использовать следующие методы ORM:

all()

get()

filter()

exclude()

lookup fields, например:

title__icontains

price__gte

price__lte

9. Reverse relation

категорию.

Студент должен показать, что понимает обратные связи в Django ORM.

10. select_related и prefetch_related

В проекте обязательно показать оптимизацию запросов:

на странице списка книг использовать select_related() для автора и категории;

на странице деталей книги использовать prefetch_related() для отзывов.

Использовать:

aggregate()
